# Robot Factory - AI Game Demo

## Overview
Claude Codeのデモ動画用プロジェクト。
ロボットが工場内のプラットフォームを移動してギアを集める2Dプラットフォーマーゲーム。

## Tech Stack
- Vite + vanilla TypeScript
- HTML5 Canvas API
- Deploy: Vercel

## Game Concept
- **キャラクター**: 工場で働くロボット（シンプルなピクセルアート風）
- **ステージ**: 工場内（金属的な背景、ベルトコンベア風のプラットフォーム）
- **目的**: プラットフォームを飛び移ってギア（歯車）を集める
- **操作**: 矢印キーで移動、スペースでジャンプ

## Demo Script (ワンショット)

### 事前準備
- リポジトリを空にし、`GAME_SPEC.md`（ゲーム仕様書）だけを配置
- 「空のディレクトリ + 仕様書1枚」の状態からスタート
- 詳細は `docs/demo/00_setup.md` 参照

### ワンショットプロンプト
```
@GAME_SPEC.md この仕様書に従って、プロジェクトのセットアップからゲームの実装まで全て行ってください。
タイトル画面、ゲームプレイ、クリア画面、ゲームオーバー画面まで全て実装してください。
実装が完了したら、AIが自動でゲームをプレイするデモモードも追加してください。
ゲームが正しく動作することを確認できたらデプロイしてください。
```

### 用途
- Webinar広告用デモ動画（約1分、倍速編集あり）
- 詳細フローは `docs/demo/03_flow.md` 参照
