# Robot Factory - AI Game Demo

## Overview
Claude Codeのデモ動画用プロジェクト。
ロボットが工場内のプラットフォームを移動してギアを集める2Dプラットフォーマーゲーム。

## Tech Stack
- Vite + vanilla TypeScript
- HTML5 Canvas API
- Deploy: Vercel

## Game Concept
- **キャラクター**: 工場で働くロボット（シンプルなピクセルアート風）
- **ステージ**: 工場内（金属的な背景、ベルトコンベア風のプラットフォーム）
- **目的**: プラットフォームを飛び移ってギア（歯車）を集める
- **操作**: 矢印キーで移動、スペースでジャンプ

## Demo Script (3 Prompts)

### Prompt 1: 基本のゲーム画面とキャラクター
```
Robot Factoryという2Dプラットフォーマーゲームを作ってください。
HTML5 Canvasを使って、800x600のゲーム画面を作成し、
ロボットキャラクター（四角いボディ、丸い目、アンテナ付き）を描画してください。
矢印キーで左右移動、スペースキーでジャンプできるようにしてください。
地面と重力の処理も実装してください。背景は工場風の暗めの色合いにしてください。
```

### Prompt 2: プラットフォームとアイテム追加
```
ゲームにプラットフォーム（足場）を複数追加してください。
足場はベルトコンベアのような見た目にしてください。
足場の上にギア（歯車）アイテムを配置して、ロボットが触れると取得できるようにしてください。
画面上部にスコア表示を追加してください。
ギアを取ったときにキラッとしたエフェクトも付けてください。
```

### Prompt 3: ゲーム性の仕上げ
```
ゲームを完成させてください。
- 障害物として電撃ビームを追加（触れるとゲームオーバー）
- ゲームオーバー画面（スコア表示 + リスタートボタン）
- 全てのギアを集めたらステージクリア画面を表示
- タイトル画面を追加（"Robot Factory" のロゴ + "Press SPACE to Start"）
```
