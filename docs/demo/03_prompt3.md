# 03. Prompt 3 — ゲーム性の仕上げ

## 目的
ゲームとしての完成度を上げ、「プロダクトとして成立するレベル」に仕上げる。
3回のプロンプトだけで遊べるゲームが完成する驚きを見せる。

## プロンプト（コピペ用）

```
ゲームを完成させてください。
- 障害物として電撃ビームを追加（触れるとゲームオーバー）
- ゲームオーバー画面（スコア表示 + リスタートボタン）
- 全てのギアを集めたらステージクリア画面を表示
- タイトル画面を追加（"Robot Factory" のロゴ + "Press SPACE to Start"）
```

## 完了後の期待状態
- タイトル画面が表示される（ゲーム起動時）
- スペースキーでゲーム開始
- 電撃ビーム障害物がある（点滅/アニメーション付き）
- 障害物に触れるとゲームオーバー画面
- ゲームオーバー画面にスコアとリスタートボタン
- 全ギア取得でクリア画面表示

## デモ時の操作ポイント
1. プロンプトを入力
2. Claude Codeが最終仕上げする様子を見せる
3. `git push` → Vercel再デプロイ
4. **タイトル画面 → ゲーム開始の流れを見せる**
5. **わざと障害物に触れてゲームオーバー → リスタートを見せる**
6. **ギアを全部集めてクリア画面を見せる**（時間があれば）

## トラブル時の対応
- タイトル画面が出ない → 「タイトル画面がスキップされる。スペースを押すまで待つようにして」
- ゲームオーバーにならない → 「電撃ビームの当たり判定を修正して」
