# 02. Prompt 2 — プラットフォームとアイテム追加

## 目的
「動くキャラだけ」の状態にゲーム要素を追加。
AIに追加指示するだけで機能拡張できることを示す。

## プロンプト（コピペ用）

```
ゲームにプラットフォーム（足場）を複数追加してください。
足場はベルトコンベアのような見た目にしてください。
足場の上にギア（歯車）アイテムを配置して、ロボットが触れると取得できるようにしてください。
画面上部にスコア表示を追加してください。
ギアを取ったときにキラッとしたエフェクトも付けてください。
```

## 完了後の期待状態
- 複数のプラットフォーム（足場）が配置されている
- 足場がベルトコンベア風の見た目
- ギア（歯車）アイテムが足場の上に配置
- ロボットがギアに触れると取得できる
- スコアが画面上部に表示される
- ギア取得時にキラッとしたエフェクト

## デモ時の操作ポイント
1. プロンプトを入力
2. Claude Codeが既存コードを読み取り → 拡張する様子を見せる
3. `git push` → Vercel再デプロイ
4. ブラウザで確認
5. **ジャンプで足場に乗り、ギアを集める操作を見せる**
6. スコアが増えることを見せる

## トラブル時の対応
- 足場に乗れない → 衝突判定の修正プロンプトを追加で入れる
- ギアが取れない → 「ギアの当たり判定を修正して」と追加指示
