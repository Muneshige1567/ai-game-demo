# 04. デモ全体フロー（タイムライン）

## デモ全体の流れ

```
[タイトル画面] → [Setup] → [Prompt 1] → [Prompt 2] → [Prompt 3] → [操作デモ] → [まとめ]
```

---

### Phase 0: 導入（〜30秒）
- 「これからAIだけでゲームを作ります」的な導入
- 画面構成の説明（左: AI / 右: デプロイ画面）

### Phase 1: Prompt 1 — 基盤構築
- プロンプト入力を見せる
- AIが動く（ファイル生成の様子）
- デプロイ → キャラが動く！
- **見せ場**: 何もない状態からゲーム画面が出現

### Phase 2: Prompt 2 — 機能追加
- プロンプト入力
- AIが既存コードを拡張する様子
- デプロイ → 足場とアイテムが追加！
- **見せ場**: ギアを集めてスコアが増える

### Phase 3: Prompt 3 — 完成
- プロンプト入力
- AIが仕上げる様子
- デプロイ → タイトル画面 + ゲーム完成！
- **見せ場**: タイトル → プレイ → ゲームオーバー → リスタートの一連の流れ

### Phase 4: 操作デモ（〜30秒）
- 実際にゲームをプレイして見せる
- 「3つのプロンプトだけで、遊べるゲームが完成しました」

---

## 注意事項
- デプロイに時間がかかる場合はトーク等でつなぐ
- エラーが出た場合は「AIに修正を指示するだけで直る」という見せ方もアリ
- 動画編集で待ち時間はカット可能
