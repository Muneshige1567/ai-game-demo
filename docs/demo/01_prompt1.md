# 01. Prompt 1 — 基本のゲーム画面とキャラクター

## 目的
何もない状態から、ゲームの土台を一気に作る。
「AIに1つのプロンプトを入れるだけで、ゲームの基盤ができる」インパクトを見せる。

## プロンプト（コピペ用）

```
Robot Factoryという2Dプラットフォーマーゲームを作ってください。
HTML5 Canvasを使って、800x600のゲーム画面を作成し、
ロボットキャラクター（四角いボディ、丸い目、アンテナ付き）を描画してください。
矢印キーで左右移動、スペースキーでジャンプできるようにしてください。
地面と重力の処理も実装してください。背景は工場風の暗めの色合いにしてください。
```

## 完了後の期待状態
- 800x600 の Canvas が表示される
- 工場風の暗い背景
- ロボットキャラクターが画面に表示される
- ← → キーで左右移動できる
- スペースキーでジャンプできる（重力で落下する）
- 地面がある

## デモ時の操作ポイント
1. プロンプトを入力（ゆっくり見せる）
2. Claude Codeが複数ファイルを生成する様子を見せる
3. 自動で `git push` → Vercel再デプロイ
4. ブラウザをリロードして画面を見せる
5. **実際にキーボードで操作して見せる**（← → で移動、スペースでジャンプ）

## トラブル時の対応
- Canvas が表示されない → ブラウザのコンソールでエラー確認
- キー入力が効かない → Canvas にフォーカスが当たっているか確認
