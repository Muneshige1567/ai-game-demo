# Robot Factory - ゲーム仕様書

## 概要
工場で働くロボットがプラットフォームを移動してギア（歯車）を集める2Dプラットフォーマーゲーム。

## 技術仕様
- HTML5 Canvas（800x600）
- Vite + TypeScript
- 外部ライブラリ不使用

## キャラクター：ロボット
- 四角いボディ、丸い目2つ、頭にアンテナ
- ピクセルアート風のシンプルなデザイン
- 操作: 矢印キーで左右移動、スペースキーでジャンプ

## ステージ
- 背景: 工場風の暗めの色合い（ダークグレー系）
- 地面: 金属的な質感
- プラットフォーム: ベルトコンベア風の足場を複数配置
- 配置は遊びやすいバランスで

## アイテム：ギア（歯車）
- 足場の上に配置
- ロボットが触れると取得
- 取得時にキラッとしたエフェクト
- 画面上部にスコア表示

## 障害物：電撃ビーム
- ステージ上に配置（水平 or 垂直）
- 点滅アニメーション付き
- 触れるとゲームオーバー

## ゲームフロー
1. タイトル画面（"Robot Factory" ロゴ + "Press SPACE to Start"）
2. ゲームプレイ（ギアを集める）
3. 全ギア取得 → ステージクリア画面
4. 障害物に触れる → ゲームオーバー画面（スコア表示 + リスタート）

## デモモード（自動プレイ）
- タイトル画面で5秒間操作がなければ自動でデモモードを開始
- AIがロボットを自律操作してギアを集める
- 障害物を回避しながらステージクリアを目指す

## 物理
- 重力処理（自然な落下）
- プラットフォームとの衝突判定
- 画面端の制限
